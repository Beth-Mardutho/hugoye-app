Forked from the syriac-corpus-app, a subset of the Syriaca.org software then rebuilt as a static site and OpeSearch index as a Gaddel app.
=======

All publications of Hugoye are made available online in a free and open format using the Creative Commons licenses.

### Dependencies
#### TEI data 
TEI data for publications is available: [https://github.com/Beth-Mardutho/hugoye-data]
The data must be upoladed to your s3 bucket instance with the file structure that maps to the desired urls in order to use the app.
